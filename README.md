# 💬 Toxic Comment Detection System

This project detects whether a given text is toxic or non-toxic using Machine Learning.

## 🚀 Features
- Text preprocessing using NLP
- TF-IDF feature extraction
- Logistic Regression / SVM model
- Streamlit web app
- Real-time prediction with confidence score

## 🧠 Model
- Trained on Hate Speech dataset
- Handles class imbalance using class_weight
- Evaluated using Accuracy and F1-score

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- NLP (NLTK)
